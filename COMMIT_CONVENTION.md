# 提交规范

建议采用 Conventional Commits 风格。

## 1. 推荐格式

```text
type(scope): subject
```

## 2. 常用类型

- `feat`
- `fix`
- `docs`
- `refactor`
- `test`
- `chore`
- `build`
- `ci`

## 3. 示例

```text
feat(router): add dynamic route bootstrap
fix(auth): handle token refresh redirect
docs(readme): add deployment guide
refactor(users): split form drawer component
```
