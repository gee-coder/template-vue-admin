# 安全说明

感谢你帮助提升模板仓库的安全性。

## 1. 适用范围

本仓库是基础模板，以下问题建议优先通过私下方式反馈：

- Token 刷新逻辑存在风险
- 权限控制、动态路由、按钮显隐逻辑存在绕过可能
- 示例配置泄露敏感信息
- 前端构建或部署默认行为存在高风险安全问题

## 2. 不建议公开披露的内容

- 可直接利用的漏洞细节
- 可访问真实环境的 Token、凭证、Cookie
- 明确的攻击步骤和利用代码

## 3. 推荐反馈方式

- 整理问题描述、影响范围、复现步骤与建议修复方向
- 不要在公开 issue 中直接披露敏感内容
- 优先联系维护者后再讨论公开修复说明

## 4. 修复策略

维护者会确认问题、评估影响并尽快修复，同时更新文档或默认配置以减少误用。
