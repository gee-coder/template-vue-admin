# 行为准则

为了让这个模板仓库适合长期协作、复用和公开讨论，请所有贡献者遵守以下约定。

## 1. 我们鼓励的行为

- 尊重不同经验水平的贡献者
- 用事实、示例和上下文讨论问题
- 给出可执行的建议，而不是只指出问题
- 默认以合作、建设性的方式交流
- 对新手问题保持耐心

## 2. 我们不接受的行为

- 人身攻击、羞辱、嘲讽
- 带有歧视、骚扰、侮辱性的表达
- 持续性的恶意争辩或破坏协作
- 泄露他人隐私、凭证、敏感配置
- 明知会破坏仓库秩序的刷屏或无关内容

## 3. 协作建议

- 提 issue 前先阅读 README、CONTRIBUTING 与提交规范
- 提 PR 时请说明目的、影响范围、验证方式
- 评审意见优先聚焦事实、风险与改进方案

## 4. 处理方式

维护者可以对不当行为进行提醒、删除内容、关闭讨论、拒绝合并，必要时限制参与。

## 5. 联系方式

若你发现严重问题，请优先通过 [`SECURITY.md`](./SECURITY.md) 或私下渠道联系维护者，而不是在公开 issue 中直接披露敏感内容。
